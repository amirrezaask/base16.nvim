## base16.lua

lua library to use [base16](https://github.com/chriskempson/base16) themes in
[Neovim](https://github.com/neovim/neovim).

## Usage

Install via your plugin manager. e.g.

```vim
use { "amirrezaask/base16.nvim" }
```
